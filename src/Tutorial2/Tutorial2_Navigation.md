# ナビゲーション作成
この章ではシート遷移・選択状態を変更するナビゲーションボタンを作成します。  
これから作成するエクステンションの機能です。

- シート一覧を表示する

- ボタンをクリックするとそのシートに遷移する

- 選択状態を進む・戻る・クリアボタンを表示する

- ボタンをクリックすると選択状態が変化する

シート遷移用のエクステンションと選択条件用のエクステンションを分けて作成しましょう。
